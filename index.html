Mijn Taken

Verbind eenmalig met GitHub om je taken overal gesynchroniseerd te houden. Volledig gratis, geen betaalgegevens nodig.

Stap 1

Ga naar github.com/settings/tokens/new en log in met je GitHub-account.

Stap 2

Geef het token een naam, zet vervaldatum op No expiration, vink gist aan. Kies Classic token.

Stap 3

Klik Generate token, kopieer de code (ghp_...) en plak hem hieronder. Hij wordt permanent opgeslagen in deze browser.
